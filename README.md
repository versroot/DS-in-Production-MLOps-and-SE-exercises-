### GitHub File Search
A command-line tool to search for specific files across GitHub repositories. See [GITHUB_FILE_SEARCH.md](GITHUB_FILE_SEARCH.md) for detailed documentation.

**Quick Start:**
```bash
go build -o github-file-search github-file-search.go
./github-file-search README.md
```
